  • Visual programming based smart actuation system for large scale IoT

    Undergraduate Thesis at Carnegie Mellon University

    GitHub repository

    Rule engine is a node-red based tool to program simple rules to actuate devices that are connected to the Building Depot IoT platform. It is a very unique way in which users can interact with the sensors deployed around them and their devices. The system employs machine learning on the sensor data collected to first filter the sensors releveant to one user, and then actuate the device associated with that user.
    Salient Features:

    • It is built on the IoT Platform GioTTo part of the Google IoT Expedition for large scale sensor deployments.
    • The system provides a very transparent and modifiable platform using Node-red to the rules made by the machine learning system.
    • It can actuate any device registered on the platform and on the generic actuation engine (for more details see).

    For more detials on the thesis please refer the thesis report.
